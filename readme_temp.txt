# Sistema de Gestión de Estudiantes - Examen Parcial## Requisitos- Node.js v18+ - Angular CLI- Cuenta en MongoDB Atlas## Ejecución del Proyecto### 1. Backend1. Entrar a la carpeta `backend`.2. Ejecutar `npm install`.3. Configurar el archivo `.env` con la variable `MONGODB_CNN`.4. Ejecutar `npm run dev` o `node index.js`.### 2. Frontend1. Entrar a la carpeta `frontend`.2. Ejecutar `npm install`.3. Ejecutar `ng serve`.4. Abrir `http://localhost:4200` en el navegador.## Funcionalidades- CRUD completo de estudiantes.- Búsqueda dinámica en tiempo real.- Paginación de resultados.
